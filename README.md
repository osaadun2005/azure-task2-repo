# azure-task2-repo
Configure and Deploy a PostgreSQL database on Azure Database for PostgreSQL– Flexible Server option, which offers high availability, scalability, and managed backups.


1. **Task Overview**

- Configure and connect a PostgreSQL database
- Owner: Intern/ Osa Adun
- Instructor: Gabriel Mekuleyi
- Status: Ongoing


2. **Perequisite**

- Microsoft Azure Platform
- Azure subscription
- Azure Portal acces
- PostgreSQL credentials (username, password, database name)
- Azure Cloudshell
- PgAdmin4
